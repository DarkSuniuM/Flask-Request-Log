# How to run this example:
  1. `git clone https://github.com/DarkSuniuM/Flask-Request-Log.git .`
  2. `python3 -m virtualenv venv`
  3. `source venv/bin/activate`
  4. `pip install -r requirements.txt`
  5. `flask run`
